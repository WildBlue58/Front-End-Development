# 学生成绩提取工具 - 测试报告

## 测试概述

本测试报告验证学生成绩提取工具在不同场景下的稳定性和准确性。

## 测试环境

- 操作系统: Windows
- Node.js 版本: 18.0+
- 浏览器: Chrome/Chromium
- 目标网站: https://172-20-130-13.atrust.ecut.edu.cn/jwglxt/cjcx/cjcx_cxDgXscj.html

## 测试项目

### 1. 功能测试

#### 1.1 成绩提取功能

**测试目标**: 验证工具能够成功从教务系统提取成绩数据

**测试步骤**:
1. 启动后端服务
2. 启动前端应用
3. 点击"提取成绩"按钮
4. 观察浏览器自动化过程
5. 检查成绩数据是否正确显示

**预期结果**:
- 浏览器成功启动并访问目标网站
- 网络请求被正确拦截
- 成绩数据被成功提取和解析
- 成绩表格正确显示数据

**实际结果**: ✅ 通过

#### 1.2 数据解析准确性

**测试目标**: 验证数据解析模块能够准确提取平时分、期末分和总分

**测试步骤**:
1. 提取成绩数据
2. 检查每门课程的平时分、期末分、总分是否正确
3. 验证数据类型和格式

**预期结果**:
- 平时分、期末分、总分都被正确提取
- 数据格式正确（数字类型）
- 缺失数据正确处理为 null

**实际结果**: ✅ 通过

#### 1.3 成绩统计功能

**测试目标**: 验证统计信息计算的准确性

**测试步骤**:
1. 提取成绩数据
2. 检查统计信息显示
3. 手动验证计算结果

**预期结果**:
- 平均平时分、期末分、总分计算正确
- 最高分、最低分识别正确
- 及格课程数统计准确

**实际结果**: ✅ 通过

#### 1.4 数据导出功能

**测试目标**: 验证数据导出为 Excel 和 CSV 格式的功能

**测试步骤**:
1. 提取成绩数据
2. 点击"导出成绩"按钮
3. 选择"导出为 Excel"
4. 验证文件下载和内容
5. 选择"导出为 CSV"
6. 验证文件下载和内容

**预期结果**:
- Excel 文件成功下载
- 文件包含所有成绩数据
- CSV 文件成功下载
- 文件格式正确，可正常打开

**实际结果**: ✅ 通过

#### 1.5 数据刷新功能

**测试步骤**:
1. 提取成绩数据

**预期结果**:
