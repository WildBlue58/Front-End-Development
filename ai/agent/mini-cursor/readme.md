## Agent 是什么？
其实就是给大模型扩展了Tool和Memory ,他本来就可以思考、 规划、你给他用tool 扩展了能力，
他就可以**自动**做事情，用memory 管理记忆， 他就可以记住你想它记住的东西，还可以使用RAG
查询内部知识来获取知识（context）。

这样一个知道内部知识、能思考规划、有记忆，能够帮你做事情的扩展后的大模型，就是一个Agent.
